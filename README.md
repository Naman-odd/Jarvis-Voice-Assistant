# Jarvis - AI Voice Assistant

A Python-based voice assistant that responds to voice commands.

## Features
- Voice recognition
- Opens websites (YouTube, Google, Instagram)
- Wikipedia search
- Tells current time and date
- Plays music
- Opens Discord, Anaconda, VS Code

## Tech Stack
- Python
- SpeechRecognition
- win32com
- Wikipedia API
- WebBrowser

## How to Run
1. Install dependencies: pip install speechrecognition wikipedia pywin32 pyaudio
2. Run: python jarvis.py
